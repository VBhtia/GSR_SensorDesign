# GSR Sensor Design

![](https://github.com/VBhtia/GSR_SensorDesign/blob/master/GSR_Circuit.png)

The complete circuit for Galvanic Skin Response Sensor consists of four stages :

1. Wheatstone Bridge 
2. Unity Buffer 
3. Differential Amplifier 
4. Low Pass Filter

| S.No | Item   | Value |
|------|--------|-------|
| 1    | R3, R4 | 10k   |
| 2    | RV1    | 50k   |
| 3    | R7, R8 | 1.3M  |
| 4    | R5, R6 | 100k  |
